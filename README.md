# TikTalk

Учебный frontend-проект, разработанный на Angular.

## Технологии

* Angular
* TypeScript
* SCSS
* HTML5
* Git

## Реализовано

* профиль пользователя;
* фильтрация контента;
* поиск;
* адаптивная верстка;
* работа с компонентами Angular;
* организация структуры приложения.

## Что изучено в процессе

* Angular Components;
* TypeScript;
* Работа с сервисами;
* SCSS;
* Git и GitHub;
* Архитектура frontend-приложений.

## Запуск проекта

```bash
npm install
ng serve
```

После запуска открыть:

```text
http://localhost:4200
```
